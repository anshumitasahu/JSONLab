# JSONLab

A modern developer utility for formatting, validating, and working with JSON data easily.

## Features

* ⚡ Fast JSON formatting
* ✅ JSON validation
* 🎨 Clean developer-focused interface
* 🌙 Dark mode experience
* 🔒 Client-side processing for privacy
